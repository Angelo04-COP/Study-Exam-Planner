# Study Planner & Exam Tracker

**Istruzioni per l'esecuzione:**

1. Assicurarsi di avere Node.js installato.
2. Aprire il terminale nella cartella del progetto ed eseguire `npm install` per scaricare le dipendenze.
3. Avviare l'app eseguendo `npx expo start`.
4. Premere `w` per avviare l'app nel browser, oppure scansionare il QR code con l'app Expo Go.
